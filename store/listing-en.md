# Chrome Web Store Listing — English

## Name
Tab Manager — Side Panel

## Short Description (132 characters max)
Organize all your tabs by domain in a side panel. Search, group, close old tabs, and manage multiple windows — all in one place.

## Detailed Description

Tired of drowning in browser tabs? Tab Manager gives you a clean, organized overview of every open tab — right in Chrome's side panel.

All your tabs are automatically grouped by website domain, sorted by most recently accessed. Find what you need instantly.

Local addresses (localhost, 192.168.x.x, etc.) show the page title instead of the raw IP. Different ports are grouped separately — localhost:3000 and localhost:8080 each get their own group. Rename any domain group for faster recognition.
